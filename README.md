
# AI Supply Chain Advisor

An AI-powered supply chain analytics system that forecasts demand and provides inventory optimization recommendations.

Features
- Demand forecasting using machine learning (XGBoost / LSTM)
- Inventory optimization (safety stock & reorder point)
- Stockout prediction
- Interactive Streamlit dashboard
- Optional AI assistant for supply chain insights

Tech Stack
Python, Pandas, XGBoost, TensorFlow, Streamlit

Run the project

pip install -r requirements.txt
streamlit run app/streamlit_app.py
